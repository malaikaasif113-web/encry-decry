# Simple Encryption and Decryption System

## Project Description

The Simple Encryption and Decryption System is a C++ console application that implements basic cryptographic techniques. The program allows users to encrypt and decrypt text using different methods including Caesar Cipher, Atbash Cipher, XOR Encryption, Vigenère Cipher, and Reverse String Encryption.

## Encryption Methods

### Caesar Cipher

Shifts each letter by a fixed number.

Example:
HELLO → KHOOR (Shift = 3)

### Atbash Cipher

Maps A to Z, B to Y, C to X, etc.

Example:
HELLO → SVOOL

### XOR Encryption

Uses a secret key and XOR operation for encryption and decryption.

### Vigenère Cipher

Uses a keyword to encrypt text.

Example:
Text = HELLO

Key = KEY

### Reverse String Encryption

Reverses the complete string.

Example:
HELLO → OLLEH
